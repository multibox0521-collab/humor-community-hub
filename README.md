# 유머 커뮤니티 모음 사이트

한국의 주요 유머 커뮤니티 사이트들을 한눈에 확인하고, **실시간 크롤링으로 실제 게시글**을 모아볼 수 있는 통합 플랫폼입니다.

## ⚡ 핵심 기능: 실시간 크롤링!

**Vercel Functions를 사용한 서버리스 크롤링**으로 실제 게시글을 가져옵니다!
- 클릭하면 **진짜 게시글**로 이동
- 5개 커뮤니티 실시간 수집
- 완전 무료!

## 🎯 주요 기능

### ✅ 현재 완성된 기능

1. **실시간 크롤링** ⚡ 🆕 NEW (Vercel Functions!)
   - **진짜 게시글을 실시간으로 크롤링!**
   - 5개 커뮤니티: 클리앙, 루리웹, 뽐뿌, 개드립, 오늘의유머
   - **클릭하면 실제 게시글로 이동** (이슈링크처럼!)
   - 서버리스 아키텍처 (Vercel Functions)
   - 자동 업데이트 (새로고침 버튼)
   - 커뮤니티별 필터링
   - 제목 검색 기능
   - 완전 무료!

2. **실시간 RSS 피드** (대체 방법)
   - RSS 제공 커뮤니티의 **실제 게시글**을 실시간으로 가져오기
   - 8개 커뮤니티: 클리앙, 루리웹, 뽐뿌, 보배드림, 개드립, 오늘의유머, SLR클럽, MLB파크
   - **클릭하면 실제 게시글로 이동** (샘플 아님!)
   - 커뮤니티별 필터링
   - 제목 검색 기능
   - 새로고침 버튼으로 최신 글 업데이트
   - 시간 표시 (방금 전, N분 전, N시간 전)

2. **인기글 모아보기** 🔥 (샘플 데이터)
   - 모든 커뮤니티의 인기글을 한 페이지에서 확인
   - 100개의 샘플 인기 게시글 제공
   - **시간대별 필터**: 전체/1시간/3시간/6시간/12시간
   - **사이트별 필터**: 전체/디시인사이드/웃긴대학/보배드림 등 10개 커뮤니티
   - **검색 기능**: 제목으로 실시간 검색
   - 한 줄 리스트 형태로 많은 게시글 표시
   - 사이트 라벨(디시, 웃긴, 보배 등) + 제목 중심의 깔끔한 레이아웃
   - 줄마다 색상 구분 (짝수/홀수)
   - 조회수, 댓글 수, 시간 표시
   - 클릭시 해당 커뮤니티의 베스트 페이지로 이동

3. **연예기사 모아보기** ⭐ (샘플 데이터)
   - 주요 언론사의 연예 뉴스를 한 페이지에서 확인
   - 100개 이상의 샘플 연예 기사 제공
   - **언론사별 필터**: 전체/네이버/스포츠조선/OSEN/디스패치 등 8개 언론사
   - **검색 기능**: 기사 제목으로 실시간 검색
   - 한 줄 리스트 형태로 빠른 정보 파악
   - 언론사 라벨 + 제목 중심 레이아웃
   - 조회수, 댓글 수, 시간 표시
   - 클릭시 해당 언론사의 연예 섹션으로 이동

4. **커뮤니티 목록 표시**
   - 16개의 주요 유머 커뮤니티 사이트 정보 제공
   - 각 사이트의 아이콘, 이름, 설명, 태그 표시

5. **베스트/인기글 바로가기**
   - 각 커뮤니티의 베스트/인기글 섹션으로 직접 이동
   - 메인과 베스트 버튼으로 원하는 페이지 선택 가능
   - 카드 클릭시 베스트 페이지로 바로 이동

6. **뷰 모드 전환**
   - 그리드 뷰: 카드 형태로 보기
   - 리스트 뷰: 목록 형태로 보기

7. **검색 기능**
   - 사이트 이름, 설명, 태그로 실시간 검색
   - 검색 결과 즉시 반영

8. **즐겨찾기 기능**
   - 자주 방문하는 사이트를 즐겨찾기에 추가
   - 로컬 스토리지에 저장되어 재방문시에도 유지

9. **방문 기록**
   - 방문한 사이트 자동 추적
   - 방문 배지 표시

10. **통계 대시보드**
    - 전체 사이트 수
    - 방문한 사이트 수
    - 즐겨찾기한 사이트 수

11. **반응형 디자인**
    - 데스크톱, 태블릿, 모바일 모두 최적화
    - 터치 친화적인 인터페이스

## 📁 프로젝트 구조

```
/
├── index.html          # 메인 페이지 (커뮤니티 목록)
├── rss.html            # 실시간 RSS 피드 페이지 ⭐ 실제 게시글!
├── posts.html          # 인기글 모아보기 페이지 (샘플)
├── news.html           # 연예기사 모아보기 페이지 (샘플)
├── css/
│   ├── style.css      # 메인 페이지 스타일
│   └── posts.css      # 인기글/연예기사/RSS 공통 스타일
├── js/
│   ├── data.js        # 커뮤니티 사이트 데이터
│   ├── main.js        # 메인 페이지 로직
│   ├── rss.js         # RSS 피드 로직 (실제 크롤링)
│   ├── posts-data.js  # 샘플 인기글 데이터
│   ├── posts.js       # 인기글 페이지 로직
│   ├── news-data.js   # 샘플 연예기사 데이터
│   └── news.js        # 연예기사 페이지 로직
└── README.md          # 프로젝트 문서
```

## 🚀 사용 방법

### 메인 페이지 (index.html)
1. 웹사이트 접속
2. **🌐 실시간 피드 버튼**: RSS로 실제 게시글 확인 (추천!)
3. **🔥 인기글 모아보기 버튼**: 모든 커뮤니티의 인기글을 한곳에서 확인 (샘플)
4. **⭐ 연예기사 버튼**: 언론사 연예 뉴스 확인 (샘플)
5. 각 커뮤니티 카드에서:
   - **🏠 메인 버튼**: 사이트 메인 페이지로 이동
   - **🔥 베스트 버튼**: 인기글/베스트 게시판으로 바로 이동
   - **카드 클릭**: 베스트 게시판으로 바로 이동
6. 즐겨찾기 별 아이콘 클릭으로 자주 가는 사이트 저장
7. 검색창에서 원하는 커뮤니티 검색
8. 그리드/리스트 뷰 전환으로 편한 방식으로 탐색

### 실시간 RSS 피드 페이지 (rss.html) ⭐ 추천!
1. **실제 게시글**: RSS로 커뮤니티의 진짜 게시글 가져오기
2. **커뮤니티 필터**: 8개 커뮤니티 선택
3. **검색 기능**: 제목으로 검색
4. **새로고침**: 최신 글 업데이트
5. **클릭시 실제 게시글로 이동** (이슈링크처럼!)

### 인기글 페이지 (posts.html)
1. **검색 기능**: 제목으로 실시간 검색
2. **시간대 필터**: 전체/1시간/3시간/6시간/12시간 선택
3. **사이트 필터**: 특정 커뮤니티만 선택해서 보기
4. **한 줄 리스트**: 100개 게시글을 빽빽하게 표시
5. **사이트 라벨**: 디시, 웃긴, 보배 등 색상별 구분
6. 게시글 클릭시 해당 커뮤니티 베스트 페이지로 이동
7. 조회수, 댓글 수, 시간 한눈에 확인

### 연예기사 페이지 (news.html)
1. **검색 기능**: 기사 제목으로 실시간 검색
2. **언론사 필터**: 특정 언론사만 선택해서 보기
3. **한 줄 리스트**: 100개 이상의 기사를 빽빽하게 표시
4. **언론사 라벨**: 네이버, 스조, OSEN 등 색상별 구분
5. 기사 클릭시 해당 언론사의 연예 섹션으로 이동
6. 조회수, 댓글 수, 시간 한눈에 확인

## 📱 반응형 지원

- **데스크톱**: 3열 그리드 레이아웃
- **태블릿**: 2열 그리드 레이아웃  
- **모바일**: 1열 레이아웃, 카드 클릭으로 사이트 방문

## 💡 주요 특징

### 샘플 데이터 방식
- **실제 API 연동 불가**: 대부분의 한국 커뮤니티는 공개 API를 제공하지 않으며 CORS 제약이 있습니다
- **샘플 게시글 제공**: 24개의 실제 같은 샘플 인기글로 UI/UX 구현
- **실제 링크 연결**: 게시글 클릭시 해당 커뮤니티의 실제 베스트 페이지로 이동
- **이슈링크 스타일**: issuelink.co.kr과 유사한 디자인과 기능

## 🔮 향후 개발 가능한 기능

1. **실시간 데이터 업데이트**
   - 서버 기반 크롤링으로 실제 인기글 수집 (정적 사이트에서는 불가)
   
2. **카테고리 필터링**
   - 게임, 스포츠, 생활 등 카테고리별 필터
   
3. **정렬 옵션**
   - 이름순, 조회수순, 댓글순, 시간순 정렬

4. **사용자 커스터마이징**
   - 사용자가 직접 게시글 데이터 추가/수정
   - 관심있는 사이트만 선택해서 보기

5. **다크 모드**
   - 어두운 테마 지원

6. **북마크 기능**
   - 관심있는 게시글 저장
   - 나중에 읽을 목록

7. **공유 기능**
   - SNS 공유 버튼
   - URL 복사
